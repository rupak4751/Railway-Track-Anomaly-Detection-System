📖 Project Overview

This project presents an automated system for detecting railway track anomalies using deep learning techniques. The system analyzes real-world railway track images to identify defects such as cracks, misalignment, missing bolts, and surface deformation.

A primary dataset was collected from multiple railway locations across Bangladesh to ensure real-world diversity and reliability.
🧠 Machine Learning Models

Several deep learning architectures were implemented and evaluated:

    Sequential CNN
    ResNet50 (Best Performing Model)
    MobileNetV2
    InceptionV3
    Hybrid CNN Model

Explainable AI techniques using Grad-CAM were applied to visualize defect regions.
📊 Performance Highlights

    Best Model: ResNet50
    Accuracy: 96%
    AUC Score: 99%
    High precision and recall for defect detection
